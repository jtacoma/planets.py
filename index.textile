h1. planets.py

This page is a demonstration of how to use a <a href="https://github.com/jtacoma/planets.py">planets.py</a> service from a web page.

For example, here is some information about where some planets are today:

bq. <div id="now" />

<script src="http://code.jquery.com/jquery-1.4.2.js"></script>
<script type="text/javascript">
  jQuery.getJSON("http://ephemeris.yellowseed.org/helio?callback=?", function(data) {
    jQuery('#now').html(
      'Moon: ' + data[0].bodies[1].lon
    );
  });
</script>

Your browser loaded this information from "ephemeris.yellowseed.org":http://ephemeris.yellowseed.org/helio but you could run your own planets.py service anywhere else, if you wanted to.  To put this in another HTML page, just copy and paste this code:
