h1. planets.py

*planets.py* is a python web application designed to publish ephemeris data.  At present, it produces only heliocentric ephemerides.

For some examples of what can be done, see the "demo":http://yellowseed.org/planets.py/.

h2. Requirements

To run planets.py on your own server, you will have to install the following:

* "Python":http://python.org v2.6 or higher, but less than v3.0
* "pyswisseph":http://pyswisseph.chaosorigin.com/
